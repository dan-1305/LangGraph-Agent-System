# Trang chủ

Chào mừng đến với hệ thống tài liệu nội bộ của **LangGraph Agent System**.

Hệ thống được thiết kế theo mô hình **Polymorphic Agent System**, hỗ trợ đa dạng các dự án con (Sub-projects) sử dụng chung một Base Agent và hệ thống RAG cốt lõi.

Vui lòng tham khảo các mục bên trái để biết thêm chi tiết.
